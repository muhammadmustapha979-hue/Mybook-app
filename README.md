# 📒 My Book Pro - Digital Ledger

A Progressive Web App (PWA) for small business owners to track sales, purchases, debts, and business analytics.

## Features

- **Daily Dashboard** - Today's profit, transactions, and active debts
- **Quick Entry** - One-tap recording for sales and purchases
- **Voice Input** - Hands-free transaction entry
- **Debt Management** - Track payments, history, and send reminders
- **Multi-Business** - Switch between different businesses
- **Analytics** - Charts, trends, and insights
- **Offline First** - Works without internet
- **Cloud Sync** - Firebase backup

## Setup

1. Replace `YOUR_API_KEY` in `index.html` with your Firebase config
2. Create `icons/` folder with icon images (72x72 to 512x512)
3. Deploy to GitHub Pages or any static host
